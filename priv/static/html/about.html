rinse

What

rinse is not a search engine. It is a collection of specific commands meant to return instantly useful answers.

rinse is a fully open source project that anyone can contribute to.

Why

There are 2 main reasons:

  • Make it useful so we can get instant relevant answers to common queries
  • Give an example of how to fully deploy and manage a web application

How

Technical details:

  • Written in Erlang/OTP using Cowboy web server
  • Hosted in AWS (Amazon Web Services)
  • Deployed using Ansible and CloudFormation

Source code:

  • Organization on GitHub: RinseOne
  • Source code for rinse.one web application: rinseweb
  • Source code for infrastructure and deployment: playbooks

Contribute

Contributions are highly encouraged.

The entire stack - front/back end code, infrastructure setup and deployment are all published openly. If you can think of ways to improve any of it, feel free to file an issue, or send a PR.

If you are not a software engineer, don't despair. Ideas on what to add next or how to make current commands better are welcome too.
