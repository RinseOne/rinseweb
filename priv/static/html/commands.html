rinse

Commands

rinse currently supports the following commands:

  • any 10-digit number: taken as Unix timestamp, converted to UTC date and time
  • any 13-digit number: taken as Unix milliseconds, converted to UTC date and time
  • Arithmetic operators followed by their arguments. Examples:
    + 1 2 3
    - 6 2 1
    * 2 3.3 8
    / 4 2 2 2
  • convert: convert between arbitrary units. Examples:
    convert 14kg to pounds
    convert 16 weeks to months
    convert 15 meter^2 to smoot^2
  • ddg: open DuckDuckGo search in a new tab or window with the given search term. Examples:
    ddg online privacy
  • ddgi: open DuckDuckGo image search in a new tab or window with the given search term. Examples:
    ddgi pictures of mt everest
  • ddgv: open DuckDuckGo video search in a new tab or window with the given search term. Examples:
    ddgv category theory
  • ddgn: open DuckDuckGo news search in a new tab or window with the given search term. Examples:
    ddgn market
  • ddgm: open DuckDuckGo maps search in a new tab or window with the given search term. Examples:
    ddgm coffee shop
  • define: find a dictionary definition for a word in English. Examples:
    define hello
    define coffee
  • md5: compute MD5 hash of the string that follows. Example:
    md5 hello there
  • now (also timestamp or unix timestamp): current unix timestamp
  • sha: compute SHA1 hash of the string that follows. Example:
    sha hello there
  • sha2: compute SHA256 hash of the string that follows. Example:
    sha2 hello there
  • uuid: generate a new UUID with every request
  • Request metadata: IP address or user agent:
    what is my ip (also wimip)
    what is my user agent (also wimua)
  • wiki: full text search for English Wikipedia. Examples:
    wiki erlang programming
  • how do I (also how to or wikihow): search English wikihow.com. Examples:
    how do I travel around he world
    how to make brownies
